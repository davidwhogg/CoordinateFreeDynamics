# Coordinate-Free Dynamics
*Is there any chance we could make things more precise by assuming less?*

## authors:
- **David W. Hogg** (NYU) (MPIA) (Flatiron)

## projects:
- Figure out *not* what Newtonian mechanics looks like, coordinate free (that's known),
  but instead what the *observational* study of stellar dynamics looks like when there is no coordinate system.
